# Kapoor & Associates Legal Platform
Full-stack legal practice management system.

## Stack
- Frontend: React 18, deployed on Vercel
- Backend: Spring Boot 3.2.1 (Java 17), deployed on Railway
- Database: PostgreSQL (prod) / SQLite (dev)
- Automation: n8n workflows

## Local Development
### Backend
cd backend-java && mvn spring-boot:run

### Frontend
cd frontend && npm install && npm start

## Environment Variables
See .env.example in each folder.
